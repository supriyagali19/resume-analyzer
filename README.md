# AI Resume Analyzer 🚀

## Features
- Resume role prediction using ML (LinearSVC)
- Semantic skill extraction using SBERT
- Skill gap analysis
- Learning recommendations
- Streamlit UI

## Tech Stack
- Python
- Scikit-learn
- NLP (spaCy, NLTK)
- Sentence Transformers
- Streamlit

## Run
```bash
streamlit run app.py
